About QuranWordAssistant

QuranWordAssistant is a lightweight web app designed for students, researchers, and readers who want to study the Qur’an more interactively.

Features

  • Search verses by keyword
  • Explore topics (AI-powered keyword grouping)
  • Read surahs with highlighted mentions of Allah and Prophets
  • Simple and mobile-friendly design

Why it was built

This project was created as a personal learning exercise in web development and AI, combining Flask, HTML/CSS, and machine learning models for semantic search. The Author can be contacted via Instagram

Disclaimer

This tool is for study purposes only. Please always refer to authentic Qur’an texts and tafsirs for deeper understanding.
